# BitsAndBytesWebsite

BitsAndBytesWebsite is a React/Vite website for the Bits and Bytes organization. It presents the group's hero section, about content, team, resources, gallery, contact area, and footer in a single polished web experience.

## Features

- Responsive landing page structure.
- Header and hero section for first impressions.
- About, team, resources, gallery, and contact sections.
- Tailwind CSS styling with a modern React component layout.
- Lucide icons for visual UI details.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Project Structure

- src/App.tsx - page composition
- src/components/Header.tsx - navigation/header
- src/components/Hero.tsx - landing hero section
- src/components/About.tsx - organization overview
- src/components/Team.tsx - team section
- src/components/Resources.tsx - resources section
- src/components/Gallery.tsx - gallery section
- src/components/Contact.tsx - contact section
- src/components/Footer.tsx - footer

## Getting Started

Install dependencies and start the dev server:

~~~bash
npm install
npm run dev
~~~

Build and preview production output:

~~~bash
npm run build
npm run preview
~~~
